## How to Use This Hub Day-to-Day

### Before a Sales Call
1. Review the relevant persona file for their company size
2. Open `discovery-questions.md` for call structure
3. Have `objection-handling.md` ready for common pushbacks
4. Know your `pricing-strategy.md` tiers

### Writing an Email
1. Check `email-marketing-guide.md` for best practices
2. Use templates from `email-sequences/`
3. Reference `outreach-templates.md` for messaging

### Creating a Proposal
1. Open `proposal-template.md` for structure
2. Include ROI calculation from `sales-playbook.md`
3. Add relevant case study from `07-Assets/case-studies/`

### Handling Objections
1. Reference `objection-handling.md` (now includes psychology insights)
2. Use the Voss Method: Label → Mirror → Calibrated Question → Silence
3. Remember: objections are buying signals

### Following Up
1. Check `proposal-followup-sequence.md` for cadence
2. Remember: 80% of sales need 5+ follow-ups
3. Best days: Thursday, Wednesday, Tuesday
