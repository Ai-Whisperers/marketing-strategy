## Lead Magnet Checklist

Before launching any lead magnet:

- [ ] Solves a real problem for the target ICP
- [ ] Provides immediate, actionable value
- [ ] Demonstrates expertise without giving away everything
- [ ] Has a clear next step (CTA)
- [ ] Landing page is conversion-optimized
- [ ] Thank you page continues the journey
- [ ] Email sequence nurtures toward sale
- [ ] Tracking is set up (downloads, conversions)

## Metrics to Track

| Metric | Target | How to Measure |
|--------|--------|----------------|
| Landing page conversion | 25-40% | Visitors → Downloads |
| Email sequence open rate | 40%+ | Opens / Emails sent |
| Click-through rate | 10%+ | Clicks / Opens |
| Lead → Call rate | 5-10% | Calls booked / Total leads |

## Best Practices

### Content
- Solve ONE specific problem completely
- Make it actionable (not just informational)
- Include quick wins they can implement today
- End with "want more? here's how we can help"

### Format
- PDFs work best for guides/checklists
- Spreadsheets for calculators/templates
- Keep it focused (5-15 pages max for guides)
- Professional design builds trust

### Promotion
- LinkedIn posts linking to landing page
- Website pop-up/banner
- Email signature link
- Mentioned in content marketing
- Paid ads (once organic is working)
