# Analytics & Tracking

## Tracking Cadence

### Daily (5 min)
- Check LinkedIn notifications
- Note any significant engagement
- Log new leads/conversations

### Weekly (30 min)
- Complete weekly KPI dashboard
- Review content performance
- Identify what's working/not

### Monthly (2 hours)
- Full monthly report
- Trend analysis
- Adjust strategy as needed

### Quarterly (Half day)
- Deep performance review
- Strategy reassessment
- Goal setting for next quarter
