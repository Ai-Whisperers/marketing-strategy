## Stats Creation Guidelines

### Before Adding a New Stat:
1. **Source it**: Where does this number come from?
2. **Document assumptions**: What calculation or data?
3. **Verify it**: Can we defend this if challenged?
4. **Track it**: Add to this document

### Stat Categories:
- ✅ **Verified**: Have data to back it up
- ⚠️ **Assumed**: Reasonable but unverified
- ❌ **Aspirational**: Where we want to be (don't use in marketing)
