# Elementos de apoyo visual — posts ParaguAI

PNG reutilizables para **componer** historias de Instagram/LinkedIn: overlays, columnas en carruseles, fondos suaves detrás de copy, etc.

**No incluyen** logo ni URL: sumá el lockup desde [`../../brand/paraguai-logo.png`](../../brand/paraguai-logo.png) según [PARAGUAI-BRAND-GUIDE.md](../../brand/PARAGUAI-BRAND-GUIDE.md).

## Convención

| Prefijo | Uso sugerido |
|---------|----------------|
| `elemento-*.png` | Ilustraciones de apoyo (IA), estilo marca oscuro + acentos |
| `stock-pexels-*.jpg` | **Fotografía real** desde Pexels; ver [ATTRIBUTIONS-PEXELS.md](ATTRIBUTIONS-PEXELS.md) |

- **Ratio:** **1:1** (~1080 px); recortá a **4:5** si el brief lo pide ([INSTAGRAM-STANDARDS-es.md](../../brand/INSTAGRAM-STANDARDS-es.md)).
- **`elemento-*`:** ilustración / render suave, sin texto legible.
- **`stock-pexels-*`:** pueden mostrar UI o entornos reales; revisá que no choquen con la guía de marca antes de publicar.
- **Antes de publicar:** revisá checklist en [`../../brand/image-prompts/WORKFLOW-pre-imagen-es.md`](../../brand/image-prompts/WORKFLOW-pre-imagen-es.md) si la pieza es campaña oficial.
