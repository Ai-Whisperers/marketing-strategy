## 1. What is VetePy?

VetePy is a **shared-infrastructure SaaS platform** that provides professional websites and veterinary management tools to independent clinics across Paraguay. One technology platform powers multiple veterinary practices with customized branding.

## 10. PERSONALIZATION PROCESS (How We Customize Each Site)

### Overview

VetePy uses a **JSON-based configuration system** that allows us to quickly customize each clinic's website without rebuilding from scratch. The client fills out a Google Form, we extract the data, generate a JSON config, and their personalized site is live in 48-72 hours.

### The Process Flow

```
CLIENT                    US                         SYSTEM
   │                       │                            │
   │  1. Fills Google Form │                            │
   │ ─────────────────────>│                            │
   │                       │                            │
   │                       │  2. Review responses       │
   │                       │  3. Generate JSON config   │
   │                       │ ──────────────────────────>│
   │                       │                            │
   │                       │  4. Deploy to Vercel       │
   │                       │<─────────────────────────  │
   │                       │                            │
   │  5. Site Live!        │                            │
   │<───────────────────── │                            │
   │  (48-72 hours)        │                            │
```
