## 5. ESCENARIOS DE PRECIO

### Escenario A: Precio actual Gs. 250,000 (~USD 33)

| Pros | Contras |
|------|---------|
| Muy competitivo vs global | Puede parecer "caro" localmente |
| Buen margen para soporte | Barrera para clinicas muy pequenas |
| Posicionamiento premium | Requiere venta consultiva |

**Break-even del cliente**: 3-5 consultas nuevas/mes

### Escenario B: Precio reducido Gs. 150,000 (~USD 20)

| Pros | Contras |
|------|---------|
| Mas accesible | Margen muy bajo |
| Menor barrera de entrada | Puede percibirse como "barato/malo" |
| Mas volumen potencial | Dificil cubrir costos de soporte |

**Break-even del cliente**: 2-3 consultas nuevas/mes

### Escenario C: Precio premium Gs. 350,000 (~USD 47)

| Pros | Contras |
|------|---------|
| Mejor margen | Barrera de entrada alta |
| Clientes mas comprometidos | Mercado mas pequeno |
| Posicionamiento exclusivo | Dificil justificar sin mas features |

**Break-even del cliente**: 5-7 consultas nuevas/mes

## 8. CONCLUSION

### El precio de Gs. 250,000/mes es CORRECTO porque:

1. **Competitivo globalmente**: 70-90% mas barato que alternativas
2. **Valor claro**: El cliente recupera la inversion con 3-5 clientes nuevos
3. **Sostenible**: Permite margen para soporte y desarrollo
4. **El mes gratis elimina friccion**: No hay riesgo para probar

### Recomendaciones:

1. **Mantener** Gs. 250,000 como precio estandar
2. **Enfatizar el ROI** en la venta: "UN cliente nuevo al mes y ya se pago"
3. **Considerar** plan basico Gs. 150,000 para consultorios muy pequenos (futuro)
4. **No bajar** el precio - mejor agregar valor con features
