```html
<div class="faq-search">
  <input type="text" 
         id="faq-search-input" 
         placeholder="Search questions..." 
         aria-label="Search FAQ">
  <button type="submit">
    <i class="icon-search"></i>
  </button>
</div>
```

### JavaScript for Search:
```javascript
// Simple FAQ search filter
document.getElementById('faq-search-input').addEventListener('input', (e) => {
  const searchTerm = e.target.value.toLowerCase();
  const faqItems = document.querySelectorAll('.faq-item');
  
  faqItems.forEach(item => {
    const question = item.querySelector('.faq-question').textContent.toLowerCase();
    const answer = item.querySelector('.faq-answer').textContent.toLowerCase();
    
    if (question.includes(searchTerm) || answer.includes(searchTerm)) {
      item.style.display = 'block';
    } else {
      item.style.display = 'none';
    }
  });
});
```

## Design Specifications

- **Spacing**: Generous padding between items
- **Hover**: Subtle background color change
- **Active**: Different bg color when expanded

### Mobile:
- Stack nicely on small screens
- Large tap targets (questions)
- Easy to read answers
- Search bar full width
