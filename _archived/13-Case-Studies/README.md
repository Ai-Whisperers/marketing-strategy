# Case Studies

> Social proof that builds trust and converts prospects into clients.

## Purpose

Case studies are your most powerful sales tool. They show real results with real clients, making it easy for prospects to envision their own success.

## How to Write a Case Study

### Step 1: Gather Information
- Client name and permission to use
- Before/after metrics
- Timeline of implementation
- Quote from decision maker
- Any visuals (screenshots, charts)

### Step 2: Use the Template
Copy `case-study-template.md` and fill in each section.

### Step 3: Review with Client
Send draft to client for approval before publishing.

### Step 4: Publish and Promote
- Add to website
- Create LinkedIn post
- Use in sales conversations
- Include in proposals

## Case Study Checklist

- [ ] Client approved use of their name/story
- [ ] Specific metrics included (not vague claims)
- [ ] Direct quote from client
- [ ] Clear before/after comparison
- [ ] Relevant to target ICP
- [ ] Call-to-action at the end

## Quick Stats to Include

| Metric Type | Example |
|-------------|---------|
| Time saved | "8 hours/week → 2 hours/week" |
| Cost reduction | "40% reduction in operational costs" |
| Revenue impact | "$50K additional revenue in 90 days" |
| Efficiency | "10x faster processing" |
| Error reduction | "95% fewer manual errors" |
