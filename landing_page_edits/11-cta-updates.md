## CTA Strategy

### Three-Tier CTA Approach:

**Primary CTAs** (High Intent):
- Lead capture or booking a call
- Target: Visitors ready to take action
- Placement: Hero, after key sections

**Secondary CTAs** (Medium Intent):
- Learn more, view case studies
- Target: Visitors still evaluating
- Placement: Throughout content

**Tertiary CTAs** (Low Intent):
- Social follows, newsletter signup
- Target: Casual browsers
- Placement: Footer, sidebar

## CTA Button Design Specifications

### Primary CTAs:
**Style**:
- Large size (48-56px height)
- Bright color (brand primary or accent)
- High contrast with background
- Bold text
- Generous padding
- Subtle shadow or border

**Example CSS**:
```css
.btn-primary {
  background: #FF6B35; /* Orange/accent color */
  color: #FFFFFF;
  padding: 16px 32px;
  font-size: 18px;
  font-weight: 600;
  border-radius: 8px;
  border: none;
  box-shadow: 0 4px 12px rgba(255, 107, 53, 0.3);
  cursor: pointer;
  transition: all 0.3s ease;
}

.btn-primary:hover {
  background: #E55A28;
  transform: translateY(-2px);
  box-shadow: 0 6px 16px rgba(255, 107, 53, 0.4);
}
```

---

### Secondary CTAs:
**Style**:
- Medium size (42-48px height)
- Outline/border style
- Brand color border
- White or transparent background
- Medium weight text

**Example CSS**:
```css
.btn-secondary {
  background: transparent;
  color: #FF6B35;
  padding: 12px 28px;
  font-size: 16px;
  font-weight: 500;
  border-radius: 8px;
  border: 2px solid #FF6B35;
  cursor: pointer;
  transition: all 0.3s ease;
}

.btn-secondary:hover {
  background: #FF6B35;
  color: #FFFFFF;
}
```

---

### Mobile Optimization:
- Full-width buttons on mobile
- Minimum 48px height (easy to tap)
- Generous spacing between buttons
- Sticky CTA bar at bottom (optional)
